# Product

## Register

product

## Users

Nuomi CLI is for developers working inside a terminal who want an AI coding agent to inspect, modify, and validate a local project. Their primary job is to understand what the agent is doing, retain control over sensitive actions, and quickly verify the outcome.

## Product Purpose

Nuomi CLI provides a focused conversational workflow for coding tasks. It should make model responses, tool activity, permission decisions, execution progress, and failures easy to distinguish without interrupting the developer's flow.

## Brand Personality

Restrained, precise, and dependable. The interaction should feel close to Claude Code: compact, calm, readable, and confident without decorative noise.

## Anti-references

Avoid raw JSON as the primary presentation, excessive colors, heavy or repeated borders, verbose status messages, ambiguous icon-only states, and chat layouts that obscure the sequence of tool execution.

## Design Principles

- Keep the task transcript easy to scan from top to bottom.
- Show the agent's action and target before implementation details.
- Make pending, running, successful, denied, and failed states unmistakable.
- Reveal detail only when it helps diagnosis or approval.
- Preserve developer control without adding unnecessary prompts.

## Accessibility & Inclusion

Use symbols together with text so meaning does not depend on color. Maintain readable contrast in common dark and light terminals, support terminals with limited color, avoid decorative motion, and keep important paths and error details available as text.
